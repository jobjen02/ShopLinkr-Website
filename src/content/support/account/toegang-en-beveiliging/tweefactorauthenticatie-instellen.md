<p>Met tweefactorauthenticatie (2FA) voeg je een extra beveiligingslaag toe aan je account. Naast je wachtwoord heb je dan ook een eenmalige code nodig om in te loggen. Zo is je account beter beschermd, zelfs als iemand je wachtwoord zou achterhalen.</p>
<h3>Wat heb je nodig?</h3>
<p>Je hebt een authenticator-app nodig op je telefoon. Voorbeelden van geschikte apps zijn:</p>
<ul>
<li>
<p>Google Authenticator</p>
</li>
<li>
<p>Microsoft Authenticator</p>
</li>
<li>
<p>Authy</p>
</li>
<li>
<p>1Password</p>
</li>
</ul>
<h3>2FA inschakelen</h3>
<ol>
<li>
<p>Ga naar <strong>Instellingen &gt; Beveiliging</strong>.</p>
</li>
<li>
<p>Bij het onderdeel <strong>Tweefactorauthenticatie</strong> klik je op <strong>Inschakelen</strong>.</p>
</li>
<li>
<p>Er verschijnt een QR-code. Scan deze met je authenticator-app.</p>
</li>
<li>
<p>Lukt het scannen niet? Je kunt ook de geheime sleutel handmatig kopiëren en invoeren in je app.</p>
</li>
<li>
<p>Vul de 6-cijferige code in die je app toont en klik op bevestigen.</p>
</li>
</ol>
<p>Dat is alles! Vanaf nu wordt er bij het inloggen om een extra code gevraagd.</p>
<h3>Hoe werkt inloggen met 2FA?</h3>
<p>Na het invoeren van je e-mailadres en wachtwoord verschijnt er een extra stap. Open je authenticator-app, lees de 6-cijferige code af en voer deze in. De code verandert elke 30 seconden, dus gebruik altijd de meest recente.</p>
<h3>2FA uitschakelen</h3>
<p>Wil je 2FA weer uitschakelen? Ga naar <strong>Instellingen &gt; Beveiliging</strong> en klik op <strong>Uitschakelen</strong> bij Tweefactorauthenticatie. Je moet hiervoor een geldige 6-cijferige code invoeren ter bevestiging.</p>
