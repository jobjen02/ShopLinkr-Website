<p>Printing labels is a crucial step in your shipping process. Below you'll find the most common problems and how to solve them.</p>
<h3>Poor print quality: labels unreadable or blurry</h3>
<p>A poorly printed label can cause problems when scanning. To fix this:</p>
<ul>
<li>
<p>Check the printer settings. Make sure you have set the correct speed and print density. For Zebra printers we recommend a speed of 2 and a density of 5.</p>
</li>
<li>
<p>Enable ZPL mode. In the printer settings within ShopLinkr you can activate ZPL support. This delivers optimal print quality for Zebra printers.</p>
</li>
</ul>
<h3>No response from the printer</h3>
<p>If your printer doesn't print at all:</p>
<ul>
<li>
<p>Restart PrintNode. Make sure PrintNode is running on your computer. Restart the program if needed.</p>
</li>
<li>
<p>Check the packing station. Make sure you have selected the correct packing station in ShopLinkr, and that it is linked to the correct printer.</p>
</li>
<li>
<p>Test a label in ShopLinkr. Try printing a test label directly from ShopLinkr. This helps determine whether the problem is with the printer or with the system.</p>
</li>
</ul>
<h3>Duplicate or unexpected prints</h3>
<p>Are labels being printed twice, or are unintended documents being printed? This can happen when multiple users have PrintNode enabled on the same computer. Check that only one user is logged in and has PrintNode running.</p>
<h3>Blank or skewed labels</h3>
<p>If the printer spits out blank labels or prints labels crooked:</p>
<ul>
<li>
<p>Calibrate the printer. Zebra printers sometimes have trouble recognizing the label layout. Refer to your printer's manual for calibration steps.</p>
</li>
<li>
<p>Check the label roll. Make sure the label roll is positioned correctly and that there is no dirt or obstruction inside the printer.</p>
</li>
</ul>
