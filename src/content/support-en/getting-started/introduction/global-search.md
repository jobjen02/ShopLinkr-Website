<p>Global search lets you search several parts of ShopLinkr at once. Instead of first navigating to the right page, you simply type your query into the search bar and get instant results across products, orders, returns and shipments.</p>
<h3>What can you search?</h3>
<p>Global search looks through the following areas at the same time:</p>
<ul>
<li>
<p><strong>Products</strong>: search by product name, SKU or EAN code.</p>
</li>
<li>
<p><strong>Orders</strong>: search by order number or customer details.</p>
</li>
<li>
<p><strong>Batches</strong>: search by batch number.</p>
</li>
<li>
<p><strong>Returns</strong>: search by return number or customer details.</p>
</li>
<li>
<p><strong>Shipments</strong>: search by reference or tracking information.</p>
</li>
</ul>
<p>Results are sorted by relevance. You will see up to 25 results.</p>
<h3>How do you use it?</h3>
<ol>
<li>
<p>Click "<strong>Search</strong>" at the top left of your navigation, or press ctrl/cmd+k.</p>
</li>
<li>
<p>Type your query. Results appear instantly as you type.</p>
</li>
<li>
<p>Click a result to jump straight to that item.</p>
</li>
</ol>
<h3>Tips for searching effectively</h3>
<ul>
<li>
<p><strong>Use specific terms</strong>: search by an EAN code or order number for an instant result.</p>
</li>
<li>
<p><strong>Partial searches work too</strong>: you do not have to type the full name or the full number.</p>
</li>
<li>
<p><strong>Results are grouped by type</strong>: you can see at a glance whether a result is a product, order, return or shipment.</p>
</li>
</ul>
