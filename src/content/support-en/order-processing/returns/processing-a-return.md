<p>Once you have received a return shipment, you can process the return in ShopLinkr. For each product, you assess its condition and decide whether it goes back into stock.</p>
<h3>Step 1: Open the return</h3>
<p>Go to <strong>Returns</strong> and click the return you want to process. You will now see the details of the return, including the products the customer is sending back. You can also scan the T&amp;T code of the return, and we will look up the return and open it directly.</p>
<h3>Step 2: Assess the items</h3>
<p>For each return item, you see which product it concerns and how many units are expected. For each item, you can do the following:</p>
<ol>
<li>
<p>Choose the <strong>return status</strong> for the item. The available options are determined by your sales channel.</p>
</li>
<li>
<p>Enter the <strong>number of units received</strong>.</p>
</li>
<li>
<p>Optionally add <strong>photos</strong> of the received product (up to 3 per item). This is useful as proof for damaged or incomplete returns, for example when you have a dispute with a customer or when you want to record internally the condition in which a product was received.</p>
</li>
</ol>
<h3>Step 3: Add back to stock</h3>
<p>For each item, you can indicate whether the product should be added back to stock:</p>
<ul>
<li>
<p>Turn on <strong>Add to stock</strong> if the product can be sold again.</p>
</li>
<li>
<p>Select the <strong>location</strong> where the product should be placed back.</p>
</li>
<li>
<p>For bundle products, you select a location for each subproduct.</p>
</li>
</ul>
<p>If you do not add the product back to stock, it will not become available for sale again.</p>
<h3>Step 4: Complete the return</h3>
<p>After you have assessed all items, the return status is updated. The sales channel is automatically notified of the processing.</p>
<p>As soon as all items have been processed, the return is given the status <strong>Handled</strong>.</p>
<h3>Good to know</h3>
<ul>
<li>
<p>Returns are currently only supported for Bol.</p>
</li>
</ul>
