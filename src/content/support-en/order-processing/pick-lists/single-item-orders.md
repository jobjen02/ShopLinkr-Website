<p>A single-item order is an order that consists of just one type of product. That product can be ordered multiple times, but the order does not contain a mix of different products.</p>
<p>For example: an order with 3x the same iPhone case is a single-item order. An order with 2x an iPhone case and 1x a charger is not.</p>
<h3>Why are single-item orders useful?</h3>
<p>Single-item orders are ideal for fast processing:</p>
<ul>
<li>
<p>There is no guesswork: you immediately know which product you need.</p>
</li>
<li>
<p>You can easily pick and pack them in bulk.</p>
</li>
<li>
<p>Less chance of mistakes while picking or packing.</p>
</li>
</ul>
<h3>Filtering on single-item orders</h3>
<p>When generating a pick list, you can choose to include only single-item orders. Go to <strong>Pick lists &gt; Generate pick list</strong> and select the option to include only single-item orders. If you like, combine this with other filters such as location, sales channel or parcel type.</p>
<p>You then get a pick list with only single-item orders, which you can process quickly one after another.</p>
