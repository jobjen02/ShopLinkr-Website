<p>When generating pick lists, you often rely on the same filter combination. Saved filters mean you no longer have to set this up every time. You save the combination once and then apply it with a single click.</p>
<h3>Saving a filter</h3>
<ol>
<li>
<p>Go to <strong>Pick lists &gt; Generate pick list</strong>.</p>
</li>
<li>
<p>Set the filters you want (sales channel, location, tags, parcel type, and so on).</p>
</li>
<li>
<p>In the top right, click <strong>Saved filters &gt; Save filters</strong>.</p>
</li>
<li>
<p>Give the filter a recognizable <strong>name</strong> (for example "Mailbox parcels webshop").</p>
</li>
<li>
<p>Click <strong>Save</strong>.</p>
</li>
</ol>
<h3>Using a saved filter</h3>
<p>When generating a pick list, your saved filters appear as quick options. Click a saved filter and all of its filter settings are applied automatically. You can still adjust the filters afterwards if needed.</p>
<h3>Managing saved filters</h3>
<p>You can also delete saved filters. Filters are visible to everyone within your company, so the whole team can follow the same workflow.</p>
<h3>Good to know</h3>
<ul>
<li>
<p>Each filter combination can only be saved once. If you try to save a filter that already exists, you will get a notification.</p>
</li>
<li>
<p>A saved filter must contain at least one active filter setting.</p>
</li>
</ul>
